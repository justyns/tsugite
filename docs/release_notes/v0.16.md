# v0.16

## 0.16.0 (2026-06-20)

### `tsu exec` for external agents

A new `tsu exec` CLI lets external coding agents (Claude Code, Cursor, Aider, etc.) reuse Tsugite skills and agents by running their code inside Tsugite's tool namespace, so skill snippets that call `get_secret()`, `http_request()`, `run()`, and friends work without a bespoke client. See the `external-agent-bridge` skill for the contract.

### Per-agent sandboxing

The daemon now applies a per-agent sandbox across all execution paths, and an agent can tighten its own sandbox via frontmatter but never loosen it.

- Frontmatter domain caps are port-aware and glob-matched, and inherit terminal policy correctly.
- Frontmatter ports are capped against the union of matching ceiling patterns (no silent port widening).
- The daemon sandbox guide moved from the README to `docs/sandbox.md`.

### Notable fixes and changes

- Doc skills can opt out of Jinja rendering with a `jinja: false` frontmatter toggle.
- History replay no longer leaks the tail of an executable `python` block as phantom assistant prose, and replayed code-execution output is truncated to match the live turn.
- Compaction summaries are no longer duplicated in the chat history.
- Claude Code provider no longer doubles streamed response text.
- Anthropic cache tokens are included in usage totals (streaming and non-streaming), and streaming usage is captured when it rides the finish chunk's `choices`.
- Model-registry prefix matching rejects unrelated variants so an unlisted model can't silently inherit a sibling's pricing.
- `read_file` honors `end_line` on its own and clamps line metadata to EOF.
- Schedules: overlapping fires are suppressed without corrupting drift metadata, and repeatedly-failing schedules are surfaced instead of failing silently.
- Compaction repoints the channel index so channel history isn't orphaned.
- Agents are given the `uploads/` path for non-inlined uploads.
- Web UI: no duplicate "Working" bubble when opening a busy session, no phantom progress-done, conversation history refetches on tab resume, follow-tail stays pinned during streaming, terminal keystrokes are forwarded to the PTY stdin endpoint, and browser back/forward switches conversations.
- Internals: an `Executor` protocol replaces the dead namespace branch in the agent runner.
- CI: the PyPI publish step is re-runnable after a partial upload.
