# v0.15

## 0.15.0 (unreleased)

### New providers

- `tsugite-codex-cli` plugin: use a ChatGPT Plus/Pro subscription as a tsugite model provider, routing through the Codex Responses API. Requires a one-time `codex login` to populate `~/.codex/auth.json`; the plugin reads and refreshes that file in place under a sidecar file lock. Models: `codex_cli:gpt-5.4`, `codex_cli:gpt-5.4-mini`, `codex_cli:gpt-5.4-nano`. Streaming, reasoning content, and per-turn token accounting all wire through. See `plugins/tsugite-codex-cli/README.md` for the token-sink caveat and recovery instructions.
